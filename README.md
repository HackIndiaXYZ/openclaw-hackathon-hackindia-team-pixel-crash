# AgroGenix - AI-Powered AgriTech Platform

A comprehensive digital agriculture solution designed to empower farmers with AI-driven insights, real-time weather intelligence, drone-based crop analytics, and direct market linkage.

## Overview

AgroGenix is a full-stack AgriTech platform that leverages artificial intelligence, drone technology, and data analytics to help farmers make better decisions about soil health, crop selection, pest prevention, irrigation, and market opportunities. Built with modern web technologies, it provides an intuitive interface accessible on desktop and mobile devices.

### Problem We Solve
- Farmers lack access to real-time soil and crop health data
- Market information is fragmented across multiple sources
- Disease detection and prevention is reactive, not proactive
- Smart farming tools are expensive and complex
- Farmers need 24/7 expert guidance in local languages

## Core Features

**How It Works:**

## Technology Stack

### Frontend
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Charts**: Recharts
- **PDF Generation**: jsPDF with html2canvas
- **State Management**: React Hooks + SWR

### Backend
- **Runtime**: Node.js (Next.js API Routes)
- **AI Integration**: Vercel AI SDK 6
- **PDF Processing**: jsPDF with canvas
- **Image Processing**: Canvas API for drone analysis

### AI Services
- **LLM Provider**: Vercel AI Gateway (default)
- **Supported Models**: Claude, GPT, Grok, Groq
- **Image Processing**: Canvas API for drone analysis
- **Voice**: Web Speech API for accessibility

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn
- Modern web browser

### Installation

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/agrogenix.git
cd agrogenix
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
```

3. **Set up environment variables (optional for AI features)**
```bash
# Create .env.local file (optional)
NEXT_PUBLIC_AI_GATEWAY=true
# Add API keys if using specific providers
```

4. **Run development server**
```bash
npm run dev
# or
yarn dev
```

5. **Open browser**
Visit `http://localhost:3000`

## Quick Start Guide

### Accessing the Platform
1. **Log in** or create your farmer account
2. **View Dashboard** - See all available features and recent alerts
3. **Navigate** using the sidebar to access specific modules

### Step-by-Step Feature Usage

#### Soil Intelligence
1. Click **Soil Intelligence** from the dashboard
2. Enter your soil test results (NPK values, pH level, moisture %)
3. Alternatively, upload a soil test report (PDF/image)
4. View detailed analysis:
   - Nutrient levels with visual indicators
   - Health score and recommendations
   - Fertilizer suggestions with specific products and quantities
5. Download PDF report for your records
6. Track changes by uploading new tests periodically

**Pro Tips:**
- Take soil samples from 5+ different areas of your field
- Get tests done at start of season and after harvest
- Share PDF reports with agricultural consultants
- Use historical data to monitor soil improvement

---

#### AI Crop Advisory
1. Navigate to **AI Crop Advisory**
2. System will automatically use your latest soil data
3. Review the ranked list of recommended crops:
   - Each shows suitability percentage (80-95%)
   - Expected yield predictions
   - Current market prices
   - Disease risk profile
4. Click any crop for detailed information:
   - Optimal planting dates
   - Water and fertilizer requirements
   - Expected harvest timeline
   - Pest and disease prevention
5. Compare 2-3 crops to make your decision
6. Save selections and get notifications at key dates

**Pro Tips:**
- Compare recommendations against your experience
- Check market prices before committing to crops
- Read disease prevention section for early action items
- Use yield predictions for financial planning

---

#### Drone Analytics
1. Go to **Drone Analytics** module
2. Prepare your field images:
   - Drone photos (10-15 minutes flight)
   - At least 100-150 feet altitude for best accuracy

3. Upload image using the upload button
4. Wait for AI processing (1-3 minutes typically)
5. Review results:
   - Overall crop health score (0-100)

6. Click "Treatment Recommendations" for specific actions
7. Download report for reference and follow-up monitoring

**Pro Tips:**
- Take drone images every 2-3 weeks during growing season
- Fly early morning or late afternoon for best light
- Ensure even field coverage across entire area
- Use reports to spot-treat instead of full field applications

## Best Practices & Expert Tips

### Maximizing Platform Value

**1. Soil Testing & Analysis**
- Conduct soil tests at least twice per year (pre-season and post-harvest)
- Take samples from 5-10 different areas of each field for accuracy
- Share PDF reports with agricultural extension officers for validation
- Track changes over years to monitor soil improvement
- Use results to optimize fertilizer spending (typically save 20-30%)

**2. Crop Selection Strategy**
- Review AI recommendations 4-6 weeks before planting season
- Compare at least 3 recommended crops before deciding
- Check market prices and demand trends alongside recommendations
- Consider your experience and comfort level with crops
- Balance high-yield crops with your typical market outlets
- Use yield predictions for financial planning and loan applications

**3. Disease Prevention**
- Monitor fields weekly using Drone Analytics (every 2-3 weeks minimum)
- Act on disease detection alerts within 24 hours
- Use preventive spraying only where indicated (not entire field)
- Document all disease observations and treatments for insurance
- Build relationship with local plant protection expert for validation
- Enable AgriCare Alerts for region-specific pest outbreaks

**4. Weather-Based Farming**
- Plan weekly operations using 7-day forecast
- Avoid pesticide spraying when rain expected within 6 hours
- Check wind speed and direction before spraying
- Adjust irrigation based on rainfall predictions
- Prepare frost protection methods when frost alert is triggered
- Use harvest timing recommendations to optimize crop quality

**5. Market Timing**
- Check prices daily during harvest planning phase
- Contact 2-3 buyers for competitive quotes (not just 1)
- Use PDF price reports during price negotiations
- Build long-term relationships with reliable buyers
- Consider transportation and handling costs in calculations
- Sell during peak demand periods (use AI Assistant if unsure)

**6. Government Support Navigation**
- Research eligible schemes 2-3 months before deadlines
- Prepare required documents in advance
- Apply for multiple eligible schemes (don't limit yourself)
- Keep copies of all submissions for your records
- Follow up on applications if status unchanged for >30 days
- Attend free government training to build skills

### Common Questions & Solutions

**Q: Which feature should I use first?**
A: Start with Soil Intelligence for baseline understanding of your land. This data enables all other recommendations. Then use AI Crop Advisory for planning, Drone Analytics for monitoring, and Weather Intelligence for operations.

**Q: How accurate are the recommendations?**
A: Our AI is trained on thousands of farmer cases and research data. Suitability scores of 80-95% mean high probability of success. However, local experience and market conditions also matter. Use recommendations to validate your thinking, not replace your judgment.

**Q: What if AI recommendation doesn't match my experience?**
A: Context matters. Discuss with AgroBot - there might be local factors (microclimate, pests, markets) affecting recommendations. Share your experience with the AI for better future recommendations.

**Q: How do I know if my soil is healthy?**
A: Soil Intelligence provides a health score. Ideal NPK ratios depend on your crop. Health score >75 is good. Track changes over time - improvement shows your practices are working.

**Q: When should I harvest for best price?**
A: Use Market Linkage to track price trends and Weather Intelligence to time quality. AI Assistant can provide crop-specific advice. Generally harvest when: crop is mature, prices are favorable, and weather supports quality.

**Q: Which government schemes apply to me?**
A: Use Government Services module to filter by location, crop type, and landholding size. Schemes change annually, so check for updates before seasons start.
